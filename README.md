This repository is my first POE assignment for my BCAD degree.

This is the first part of the assignment which is a Login/Sign up code.
The entire project is a messaging system simular to Whatsapp.

It takes the information when the user signs up and adds it to a text file called "Users.txt".
When the user tries to log in it will search the same "Users.txt" file for a matching username and password

When the user signs up, it checks that the username, password and phone number are correctly formatted.
If all the information is properly formatted it will write the information to the text file.
